# medit8z-store-link

Forum-friendly redirector for the Medit8z Tebex store. Tebex's Cloudflare blocks
link-preview scrapers (403), so this page hosts the OG meta tags and bounces
visitors to the real store.

Live URL: <https://crp-live.github.io/medit8z-store-link/>
Redirects to: <https://medit8z-scripts.tebex.io/>
